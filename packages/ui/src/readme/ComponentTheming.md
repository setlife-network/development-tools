# Component Theming

<!-- STORY -->

All components can be globally customized by defining a theme property for each component.

```js
const theme = {
    Text: {
        letterSpacing: '1px',
        color: '#9b59b6'
    },
    Card: {
        padding: '2rem',
        backgroundColor: 'white',
        borderRadius: '4px',
        boxShadow: '0px 1.5px 8px rgba(0,0,0,0.07)'
    }
}
```
